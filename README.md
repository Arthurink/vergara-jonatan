## Objetivos y expectativas de la materia
- Profundizar en JavaScript, pasando de lo básico a aplicaciones más estructuradas.
- Aplicar la programación orientada a objetos y buenas prácticas en proyectos web reales.
- Diseñar interfaces con HTML/CSS y manejar eventos con JavaScript.
- Construir aplicaciones completas en la web (utilidades, mini‑juegos o simuladores simples).
- Fortalecer la lógica y la organización del código mediante clases, métodos y componentes reutilizables.
